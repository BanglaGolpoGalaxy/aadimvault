# আদিম ভল্ট (Adim Vault / AadimVault)

**A Reliqium — Where Forgotten Objects Live Again**

পৃথিবীর আদিম গল্প আর বিস্মৃত স্মৃতিচিহ্নের ডিজিটাল ভাণ্ডার।

এখানে সংগ্রাহক ও সাধারণ মানুষ তাদের পুরনো জিনিস ও স্মৃতিচিহ্ন গল্পসহ সংরক্ষণ করতে পারে, বিক্রি করতে পারে, কিংবা নিলামে তুলতে পারে। দর্শনার্থীরা এই রহস্যময় ভল্টে প্রবেশ করে আবিষ্কার করবে পৃথিবীর নানা প্রান্তের ইতিহাস।

## 🌟 মূল ফিচার (ফেজ-১)
- **পণ্য আপলোড:** ছবি, গল্প, বিক্রি/প্রদর্শনী/নিলাম
- **গল্প সংরক্ষণ:** প্রতিটি পণ্যের পেছনে সমৃদ্ধ ইতিহাস
- **কেনাকাটা ও নিলাম:** সরাসরি কেনা বা নিলামে অংশগ্রহণ
- **ডার্ক/লাইট মোড, ভাষা টগল, গুগল ট্রান্সলেট**

## 🛠️ প্রযুক্তি
| স্তর | প্রযুক্তি |
|------|-----------|
| ফ্রন্টএন্ড | HTML + CSS + Vanilla JS (GitHub Pages) |
| ব্যাকএন্ড | Node.js + Express.js (REST API) |
| ডাটাবেস | SQLite (sql.js) |
| ছবি স্টোরেজ | `/uploads` ফোল্ডার, static serve |
| ডেপ্লয়মেন্ট | Termux + Cloudflare Tunnel |
| অথেনটিকেশন | JWT (jsonwebtoken + bcryptjs) |

## 🗺️ ভিশন (তিন ফেজ)
- **ফেজ-১ (বর্তমান):** সরল ই-কমার্স + নিলাম + প্রদর্শনী + গল্প
- **ফেজ-২ (ভবিষ্যৎ):** গ্যামিফিকেশন, নিজস্ব জাদুঘর
- **ফেজ-৩ (ভবিষ্যৎ):** ভার্চুয়াল ওয়ার্ল্ড, জমি-বাড়ি, ঘুরে দেখা
